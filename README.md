## 官网前台项目

这是用于官网静态演示的独立前台项目。

### 入口页

- `index.html`：官网首页
- `rates.html`：实时汇率
- `services.html`：服务与优势
- `partners.html`：合作商家
- `partner-detail.html`：商家详情
- `academy.html`：新手学院首页
- `academy-list.html`：教程列表页
- `academy-detail.html`：教程详情页
- `about.html`：关于我们
- `support.html`：客服聊天页

### 说明

- 当前项目仅包含前台静态页面。
- 不包含后台配置页面与字段标注页面。
- 共用资源文件为 `styles.css`、`site.js`、`brand-logo.svg`、`assets/`。
